# The Mighty Path

Hello Adventurer, in this game you will be the invocated protagonist and you have to form a group, explore, find equipments and fight monsters to progress

### classes

- Knight : Base Attack = 25, Attack growth per level = 5, Base HP = 60, HP growth per level = 6, Base Mana = 5, Mana growth per level = 2, Base Magic = 5, Magic growth per level = 1
- Mage : Base Attack = 10, Attack growth per level = 1.5, Base HP = 30, HP growth per level = 3, Base Mana = 50, Mana growth per level = 6, Base Magic = 40, Magic growth per level = 5
- Cleric : Base Attack = 5, Attack growth per level = 1, Base HP = 40, HP growth per level = 4, Base Mana = 60, Mana growth per level = 10, Base Magic = 15, Magic growth per level = 3
- Rogue : Base Attack = 35, Attack growth per level = 6, Base HP = 45, HP growth per level = 5, Base Mana = 4, Mana growth per level = 2, Base Magic = 3, Magic growth per level = 1

### EXP System

- Gain Stats based on class
- gain abilities based on level and class

### Equipments

- Shop System
- Loots
- Consumables
- Crafting System (Maybe)

### game loop

- Battle System: turn based on speed
- World levels
- Different creatures/monsters
- Different NPC
